# Summary
You need:
- 1x M4x20mm bolt plus nut (or M3)
- 1x M4x8mm bolt
- 2x T-nuts for 2020 

There's an optional STL file for better mounting on the 2020 extrusion. 

Various lengths available: 10cm, 12cm, 15cm, 18cm and 20cm.
